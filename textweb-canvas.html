How to Use

  • Add Node: Enter text and click "Create Node" to add a new node to the canvas
  • Add Layer: Create layered backgrounds for visual depth
  • Customize: Adjust colors and sizes using the input fields
  • Export: Save your canvas as JSON to share or edit later
  • Import: Load a previously exported JSON file
  • Clear: Remove all nodes and layers to start fresh

Features

  • Drag nodes to reposition them on the canvas
  • Click nodes to edit their text
  • Right-click nodes to delete them
  • Support for multiple layers with different colors
  • Real-time statistics showing node and connection counts
  • Import/export for sharing your creations
