BitNet Inference Demo

About BitNet

BitNet is a 1-bit LLM inference framework that uses extreme quantization (1-bit per weight) to dramatically reduce memory footprint while maintaining reasonable accuracy. This demo explores the inference mechanics with model parameters and example outputs.

Model Parameters

Model Size Qwen 3.5-9B (1-bit quantized)
Context Window 32K tokens
Memory Reduction ~95% vs FP16
Inference Speed 2-3x faster than FP16
